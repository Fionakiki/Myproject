# Picture Editor 
<br>

### 项目平台
* PC端
### 问题描述
拍照已经成为了现代生活的一部分，吃了好吃的东西要拍照，看到好看的风景要拍照，出去旅游要拍照......
为了探索怎样使照片更加好看，我们学习了并自己创建了一个图片编辑器项目。

### 项目描述
分为四个页面：
1.主页面：照片的展示
2.关于页面：对我们项目的简单描述
3.编辑页面：对照片进行编辑
4.联系页面：对我们创建项目人的联系
